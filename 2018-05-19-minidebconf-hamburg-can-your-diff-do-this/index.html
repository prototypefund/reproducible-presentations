Getting started

try.diffoscope.org

$ sudo apt install trydiffoscope

$ trydiffoscope /path/to/file1 /path/to/file2 [--webbrowser]

      or

$ trydiffoscope --webbrowser /path/to/file1 /path/to/file2

$ sudo apt install diffoscope --install-recommends=no

$ diffoscope /path/to/file1 /path/to/file2
